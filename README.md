# EMT

写真投稿SNSアプリです。<br>
学んだことをアウトプットするために作成しました。<br>
写真を映させるためにデザインはシンプルにしました。<br>
そして少し遊び心を加えました。

## 使用技術

フロントエンド<br>
jquery-rails (4.4.0)<br>
HAML <br>
Sass<br> 

バックエンド<br>
ruby  (2.6.6)<br>
rails (6.1.1)<br>

インフラ<br>
pg (1.2.3)<br>
AWS S3<br>

**ログイン・ログアウト機能**<br>
**画像投稿・削除機能**<br>
